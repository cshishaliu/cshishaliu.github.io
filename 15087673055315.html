两道关于向量定比分点的题

题 1

已知条件可变形为
\[
(1+\alpha-\beta) \vec{PA} + \alpha (\vec{PB} - \vec{PA}) = \beta (\vec{PC} - \vec{PA})
\]
即 \((1+\alpha-\beta) \vec{PA} + \alpha \vec{AB} = \beta \vec{AC}\), 故
\[
(1+\alpha-\beta) \vec{PA} = - \alpha \vec{AB} + \beta \vec{AC}
\]
由于 \(\vec{AB}\) 与 \(\vec{AC}\) 共线, 而它们与 \(\vec{PA}\) 都不共线, 故只能是
\[
(1+\alpha-\beta) \vec{PA} = \beta \vec{AC} - \alpha \vec{AB} = \vec 0
\]
因而 \(1 + \alpha - \beta = 0\), \(\alpha - \beta = -1\).

\vec{OM} - \left( -\dfrac23 + 3 + m \right) \vec{OM} = -\dfrac23 (\vec{OA} - \vec{OM}) + 3 (\vec{OB} - \vec{OM}) + m (\vec{OC} - \vec{OM})
\]
即
$$

  • \left( \dfrac43 + m \right) \vec{OM} = -\dfrac23 \vec{MA} + 3 \vec{MB} + m \vec{MC}
    $\(
    注意到上式等号右边的向量一定在平面 \)ABC\( 内, 而左边的 \)\vec{OM}\( 不在该平面内, 故两边都必须等于零向量. 因此 \)\dfrac43 + m = 0\(, \)m = \dfrac43$.

关于向量的一个重要结论

向量中, 关于三点共线, 有如下结论:

在已知 \(\vec{PA} = \lambda \vec{PB} + \mu \vec{PC}\) 的前提下,
\(A,B,C\) 三点共线的充分必要条件是\(\lambda + \mu = 1\).

此结论不仅对于平面中的三点共线是成立的, 对于空间中的三点共线也成立.

另外, 对于空间中的四点共面问题也有一个极为类似的结论:

在已知 \(\vec{PA} = \lambda \vec{PB} + \mu \vec{PC} + \nu \vec{PD}\) 的前提下,
\(A,B,C,D\) 三点共面的充分必要条件是\(\lambda + \mu + \nu = 1\).

用这两个结论解上面两道题, 几乎可以秒解 (自己试试看?).

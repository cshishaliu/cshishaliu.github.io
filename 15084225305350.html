《奥数教程》1上的一个不详细到诡异的答案

原题

原答案

这道题的这个答案貌似“省略”了很多话.

首先是, 不妨设 \(a_1 \le a_2 \le \dots \le a_n\), 于是 \(A_1 \ge A_2 \ge \dots \ge A_n\).

然后是, 假设 \(b_1, b_2, \dots, b_n\), 按从大到小的顺序重排为 \(c_1, c_2, \dots, c_n\).

那么排序不等式告诉我们
\[
\sum b_i A_i \le \sum c_i A_i
\]

按照题意, \(\{c_i\}\) 中的最大项 \(c_1 \le p\). 假设其前 \(k-1\) 项相加不大于 \(p\), 但再加上 \(k\) 项就大于 \(p\), 即
\[
c_1 + c_2 + \dots + c_{k-1} \le p < c_1 + c_2 + \dots + c_{k-1} + c_{k}
\]
可以把 \(c_{k}\) 拆成 \(c'+c''\), 使得 \(c_1 + c_2 + \dots + c_{k-1} + c' = p\), 于是 \(c'' + c_{k+1} + \dots c_n = 1-p\).

那么
\[
\begin{aligned}
\sum c_i A_i &= c_1 A_1 + c_2 A_2 + \dots + c_k A_k + \dots + c_n A_n \\
&= (c_1 A_1 + c_2 A_2 + \dots + c' A_k) + (c'' A_k + \dots + c_n A_n) \\
&\le (c_1 A_1 + c_2 A_1 + \dots + c' A_1) + (c'' A_k + \dots + c_n A_k) \\
&= (c_1 + c_2 + \dots + c') A_1 + (c'' + \dots + c_n) A_k \\
&= p A_1 + (1-p) A_k \le p A_1 + (1-p) A_2
\end{aligned}
\]

第 (1) 问

第 (1) 问应该还比较简单, \(a=1\) 时, 函数 \(f(x) = \sqrt{x^2+1} - x\).

这个函数在 \(x \ge 0\) 时, \(\sqrt{x^2+1}\) 跟 \(x\) 的单调性趋势相同, 因此它们的差并不能直接判断单调性. 但是可以做“分子有理化”的变形:
\[
f(x) = \dfrac{\left( \sqrt{x^2+1} - x \right) \left( \sqrt{x^2+1} + x \right)}{\sqrt{x^2+1} + x} = \dfrac1{\sqrt{x^2+1} + x}
\]
那么就能明显看出来是单调减了. 需要注意的是, 以上的讨论只对 \(x \ge 0\) 的情况成立.

在 \(x \le 0\) 时, 这个函数的单调性其实是非常好讨论的, 因为这个时候, \(x\) 越大, \(\sqrt{x^2+1}\) 和 \(-x\) 都会越小, 于是 \(f(x) = \sqrt{x^2+1} + (-x)\) 当然也就越小.

于是综上所述, \(f(x) = \sqrt{x^2+1} - x\) 在整个实数集 \(\mathbb R\) 上都是单调递减的.

第 (2) 问

第 (2) 问的难度可能相对就较大了.

首先有了第一问的结论，应该很容易判断: 当 \(a \ge 1\) 的时候肯定不行; 另外也应该不难看到 \(a \le 0\) 的时候肯定可以. 所以最后求出的 \(a\) 的取值范围肯定包含所有小于等于 \(0\) 的数, 但是一定不包含任何一个大于等于 \(1\) 的数.

所以看上去, 接下来的问题在于 \((0,1)\) 区间中的值 \(a\) 能不能取? 或者说, 哪些能取, 哪些不能取? 但是想到这里, 继续往下也并没有特别好的办法.

接下来我们从另外一种思路入手. 函数 \(f(x) = \sqrt{x^2+1} - ax\) 可以看作是函数 \(y = \sqrt{x^2+1}\) 和 \(y = -ax\) 的叠加. 注意到这样一个问题1:

\(y = \sqrt{x^2+1}\) 是偶函数, 在 \(x>0\) 和 \(x<0\) 时的单调性正好相反, 于是知道偶函数在 \(x=0\) 附近的变化趋势大致是“平”的.
因此, 原来的函数 \(f(x)\) 在 \(x=0\) 附近的变化趋势完全由它的奇函数部分 \(y = -ax\) 决定.

但是, 题目中要求, 只要 \(x>0\), \(f(x)\) 就得单调增, 因此这个“奇函数分量” \(y=-ax\) 不可以是单调减. 这也就意味着必须有 \(-a \ge 0\), 也就是 \(a \le 0\).

这个结论目前看可能暂时觉得怪怪的, 但是我们确实可以尝试验证一下它. 而目前我们进行验证的唯一手段, 就是采用单调性去证明.

假设 \(x_1\) 和 \(x_2\) 是取自 \(\mathbb R^+\) 上的两个不同的数, 如果我们希望 \(f(x)\) 单调增, 那么就要求无论 \(x_1, x_2\) 怎么选取, 只要满足 \(0<x_1<x_2\) 这个前提, 都得有 \(f(x_1) - f(x_2) < 0\).

可以先做下面的计算:
\[
\begin{aligned}
f(x_1) - f(x_2)
&= \left( \sqrt{x_1^2+1} - a x_1 \right) - \left( \sqrt{x_2^2+1} - ax_2 \right) \\
&= \left( \sqrt{x_1^2+1} - \sqrt{x_2^2+1} \right) - a(x_1 - x_2) \\
&= \dfrac{x_1^2 - x_2^2}{\sqrt{x_1^2+1} + \sqrt{x_2^2+1}} - a(x_1 - x_2) \\
&= \left( \dfrac{x_1 + x_2}{\sqrt{x_1^2+1} + \sqrt{x_2^2+1}} - a \right) (x_1 - x_2)
\end{aligned}
\]
这里的最后结果中, 第二个括号里的值已经小于 \(0\), 所以我们也就希望, 只要满足 \(0<x_1<x_2\) 这个前提, 第一个括号里的数都得是正的.

但是仔细思考一下, 应该可以发现在 \(a>0\) 的时候, 这一点就是做不到. 因为在 \(a>0\) 的时候, 我总能把 \(x_1, x_2\) 都取得特别小, 特别接近于 \(0\), 从而使得 \(\dfrac{x_1 + x_2}{\sqrt{x_1^2+1} + \sqrt{x_2^2+1}} < a\).

具体来说, 只要 \(x_1\) 和 \(x_2\) 都小于 \(a\) 就有
\[
\dfrac{x_1 + x_2}{\sqrt{x_1^2+1} + \sqrt{x_2^2+1}} < \dfrac{x_1 + x_2}{\sqrt{1} + \sqrt{1}} < \dfrac{a+a}2 = a
\]

以上, 我们相当于证明了当 \(a>0\) 时, 函数 \(f(x)\) 在 \((0,a)\) 上单调递减2.

总结一下, 在第 (2) 问里面, 我们能够证明下面两点:

  1. 在 \(a\le 0\) 时, 函数 \(f(x) = \sqrt{x^2+1} - ax\) 在 \(\mathbb R^+\) 上单调递增;
  2. 在 \(a > 0\) 时, 函数 \(f(x)\) 在区间 \((0,a)\) 上单调递减, 因而也就不可能在 \(\mathbb R^+\) 上单调递增.

综合这两点, 我们就可以确定这里问里面所要求的 \(a\) 的取值范围就是 \((-\infty,0]\).
